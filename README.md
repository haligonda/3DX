# 3DX

Find Trending AI Research Papers On HuggingFace

## Overview

**3DX** is a Python-based project designed to help users discover trending AI research papers hosted on Hugging Face. Whether you're looking for the latest breakthroughs or popular topics in the AI community, 3DX provides a convenient way to stay up-to-date.

## Features

- Fetches trending AI research papers from Hugging Face.
- All AI research papers in .md format
- Simple and easy-to-use interface.
- 100% Python implementation.

## Getting Started

### Prerequisites

- Python 3.7 or above
- uv

### Installation

Clone this repository:
```bash
git clone https://github.com/haligonda/3DX.git
cd 3DX
```

Install required packages (if any):
```bash
playwright install
uv pip install crawl4ai
```

### Usage

Run the main script to find trending papers:
```bash
uv run main.py
```

Locate `.data` folder to explore papers.
